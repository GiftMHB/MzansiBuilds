#  Project Overview

##  Project Understanding

###  Problem Statement
Developers often build in isolation and lack:
- A public space to share progress  
- Easy collaboration requests  
- Celebration of completed projects  

---

##  MzansiBuilds

MzansiBuilds is a web platform where developers can:

- Create and manage accounts  
- Post projects with stage & support needs  
- View a live feed of other developers’ projects  
- Comment and request collaboration  
- Update project milestones  
- Celebrate completed projects on a **Celebration Wall**  

---

##  Purpose

MzansiBuilds aims to:
- Encourage collaboration among developers  
- Provide visibility for ongoing work  
- Motivate completion through recognition  
- Build a strong developer community  

---

##  Tech Stack

| Category            | Technology            | Justification |
|--------------------|----------------------|--------------|
| Backend            | Node.js + Express    | Lightweight, fast, great for REST APIs |
| Database           | PostgreSQL           | Reliable, supports complex queries |
| Authentication     | JWT + bcrypt         | Secure, stateless, industry standard |
| Frontend           | React + Tailwind CSS | Component-based, easy styling |
| Version Control    | Git + GitHub         | Required for collaboration |
| Testing (TDD)      | Jest + Supertest     | Ensures backend reliability |
